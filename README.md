# Python Mini Projects 🐍

A collection of Python programs built to practice core programming concepts.

## 📁 Projects Included:

### 1. 🌐 API Integration
- Fetches real-time data using Python requests library
- Demonstrates how to connect to external APIs and handle JSON responses
- Tech: Python, REST API, Requests Library

### 2. 👑 N-Queens Problem
- Solves the classic N-Queens puzzle using backtracking algorithm
- Places N queens on an N×N chessboard with no two queens attacking each other
- Tech: Python, Recursion, Backtracking, Algorithm Design

### 3. 🎯 Number Guessing Game
- Interactive CLI game where user guesses a randomly generated number
- Includes hints (too high / too low) and attempt counter
- Tech: Python, Random Module, Loops, Conditionals

### 4. 📝 Blog System (blog.py)
- Simple blog management system
- Tech: Python, OOP

### 5. 🧮 Calculator (cal.py)
- Basic arithmetic calculator
- Tech: Python, Functions

### 6. 🔢 LCM Calculator
- Calculates Least Common Multiple
- Tech: Python, Math Logic

## 🛠️ Tech Stack:
- Language: Python 3
- Concepts: API Integration, Algorithms, OOP, Game Logic, Math
